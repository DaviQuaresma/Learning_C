## Compilador - GCC para compilar C

O GCC é um compilador de linguagem C. O compilador é o responsável por transformar 
todo o código que escrevemos em linguagem C para código de máquina (afinal, o 
computador não entende a linguagem C diretamente).

## Memoria

Em C é possivel utilizar uma variavel que não foi inicializada ainda, o problema é que
o compilador não vai avisar e isso traz lixo de codigo que sobrou da ultima execução do codigo.
